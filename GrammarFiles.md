# Grammar Files
Grammar Files are used to define a Grammar in a File reather than just cerate them with code.

## Syntax
### File Over All
```
[Token Definitions]
[Grammar]
```
	 

## Example

    Grammer.build("String").matchToken(Tokens.STRING_DELIMITER).anyExcept()
